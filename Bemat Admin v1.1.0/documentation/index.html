Overview

Bemat Admin is a professional, responsive and multipurpose HTML5/CSS3 Material Design admin template powered with Twitter Bootstrap 3.3.6 Framework and jQuery library. Bemat can be used for any type of web applications: custom admin panels, admin dashboards, CMS, CRM, SAAS and more. Bemat has a sleek, clean and intuitive material design style, and has a huge collection of plugins and UI components and works seamlessly on all major web browsers, tablets and phones.

Components

Bemat is based on Twitter Bootstrap 3.x and utilize/customize almost all of the components from this framework.

All the Bemat components function the same way like Bootstrap's way, so the only thing you have to do is check the Bootstrap examples for more details.

Exclusive Plugins

Bemat utilizes eight exclusive plugins, exclusively developed by our dev team to fit Material Design.

Plugin Description
